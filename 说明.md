### 小程序技术参考

#### 本地存储value  字段说明

+ Userdata  用于存储用户的token、refresh_token、用户昵称及头像；
+ isLogin  用于存储用户当前登录状态，true为登录，false为未登陆；

#### 返回Code详解

| 序号 | Code | 详细说明                                                     |
| :--- | ---- | ------------------------------------------------------------ |
| 1    | 101  | 提交的该类型订单，有未完成的情况，该类订单必须完成后才可以继续申请 |
